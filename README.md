# tiedan

铁蛋的 OpenClaw 工作区仓库。

## 这是什么

这是一个用于保存 OpenClaw 工作区内容的 GitHub 仓库，主要包括：

- 助手身份配置
- 用户偏好与本地约定
- 日常记忆文件
- 工作区规则与运行说明

## 当前已纳入版本控制的内容

- `IDENTITY.md`：助手身份
- `USER.md`：对用户的基本了解
- `memory/`：按日期记录的记忆
- `AGENTS.md`：工作区行为约定
- `SOUL.md`：助手的个性与行为基调
- `TOOLS.md`：本地环境备注
- `HEARTBEAT.md`：心跳任务占位文件

## 不纳入仓库的内容

以下内容默认不会提交：

- `.openclaw/` 运行时数据
- 临时文件、日志文件
- `node_modules/` 等依赖目录

## 仓库用途

这个仓库主要用于：

1. 保存工作区配置
2. 跟踪初始化和后续变更
3. 让工作区内容可以同步到 GitHub
4. 为后续扩展更多自动化文件留位置

## Git 分支

- 默认分支：`main`

## 备注

这是一个偏个人化的工作区仓库，后续内容会随着实际使用逐步长出来。
